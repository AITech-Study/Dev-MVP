## 커밋 컨벤션
### 커밋 구조
- head 형식: 커밋 타입: 제목
- body 형식: 해당 내용에 대해 간결하게 설명

### 커밋 타입
- feat: 새로운 파일/폴더 추가
- fix: 파일/폴더 코드 수정
- docs: 문서 추가/수정
- test: 테스트 코드 작성
- refactor: 리팩토링 작업
- remove: 불필요한 파일 삭제
- issue: 문제가 발생한 사항 or 문제의 원인 파악 필요
