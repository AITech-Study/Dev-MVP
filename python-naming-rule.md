## 파이썬 명명 규칙

### 1. 프로젝트 및 패키지 이름
- 소문자
- 밑줄( _ ) 사용X
- 간결하고 명화하게
- 복합어가 필요시 하이픈(-) 사용

ex) project-data

### 2. 모듈(파일) 이름
- 소문자
- 밑줄( _ ) 사용O
- 간결하고 명화하게
  
ex) change_funtion.py, wordsort.py (.py는 자동으로 붙음)

### 3. 클래스 이름
- 카멜 표기법을 사용(각 단어의 첫 글자를 대문자)
  
ex)DataList, UserProfile

### 4. 변수 및 함수 이름
- 소문자 사용
- 밑줄( _ ) 사용 O
- 명확하고 구체적인 이름 사용
  
ex) total_count, process_data()

### 5. 상수 이름
- 모두 대문자 작성
- 밑줄( _ ) 사용 O
  
ex) MAX_RETRIES
